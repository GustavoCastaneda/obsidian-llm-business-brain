# Investor Intel

Use this skill for investor calls, fundraising prep, investor updates, and fundraising narrative work.

## Workflow

1. Read the relevant investor source notes.
2. Extract investor feedback, objections, interests, requested materials, and follow-ups.
3. Update `02-Business/Fundraising/Fundraising Narrative.md`.
4. Update `02-Business/Finance/Financial Model Notes.md` only when financial assumptions are mentioned.
5. Update [[MOC - Investors]] and [[MOC - Fundraising]].
6. Append an entry to `log.md`.

## Output Options

Save durable outputs in:

- `04-Outputs/Investor-Updates/`
- `04-Outputs/Briefings/`
- `04-Outputs/Strategy-Memos/`
