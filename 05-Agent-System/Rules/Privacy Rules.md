# Privacy Rules

Use these rules before preserving or sharing information.

1. Store only business-relevant personal details.
2. Never store credentials, OAuth tokens, API keys, secrets, or private connector configuration.
3. Prefer lightweight source notes with links over copying full emails, transcripts, or documents.
4. Preserve a full local copy only when the user requests it or a specific workflow requires it.
5. Ask before preserving especially sensitive personal, legal, HR, or financial information beyond a necessary summary.
6. Do not change permissions on external source documents.
7. Do not move private content into a public repository.
