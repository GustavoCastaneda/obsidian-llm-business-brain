# First Session Prompt

Copy this into a fresh Codex or Claude Code session opened from the vault root:

```text
Use this Obsidian business vault. Read AGENTS.md, START_HERE.md, index.md, and log.md. Then use business-vault-onboarding to run a first founder interview.

Ask one question at a time. Start with:

1. my name
2. company name
3. one-sentence company description
4. ICP
5. product status
6. GTM channels
7. current priorities
8. key risks
9. important sources or tools to connect

After each answer, update the right pages in 02-Business, update index.md if needed, and append to log.md.

Do not invent missing business facts. If something is unclear, add it as an open question.
```
